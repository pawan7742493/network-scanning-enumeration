# Network Scanning & Enumeration

## Objective

Performed network reconnaissance and service enumeration in a controlled cybersecurity lab.

## Lab Environment

Attacker:
- Kali Linux

Target:
- Metasploitable2

## Tools Used

- Nmap
- Enum4linux
- SMBClient

## Methodology

1. Host Discovery
2. Port Scanning
3. Service Enumeration
4. Vulnerability Identification


## Important Commands


nmap -sn target_ip


nmap -sV -sC target_ip


enum4linux -a target_ip


## Findings

- Identified open ports
- Detected running services
- Collected system information


## Learning Outcome

Improved practical knowledge of network reconnaissance used in cybersecurity.
